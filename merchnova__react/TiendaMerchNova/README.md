# Inicio

Este proyecto consiste en el desarrollo de una aplicación web full stack de comercio electrónico enfocada en la venta de productos personalizados (camisetas, tazas, sudaderas, accesorios y otros artículos de merchandising).

La plataforma permite a los usuarios registrarse, iniciar sesión, explorar el catálogo de productos, añadir artículos al carrito, realizar pedidos y completar pagos de forma segura. Además, cuenta con un sistema de gestión de pedidos y autenticación basada en tokens.

El objetivo principal es aplicar buenas prácticas de desarrollo backend y frontend, integrando autenticación segura, persistencia en base de datos no relacional y pasarela de pagos.
